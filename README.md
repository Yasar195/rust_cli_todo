# ✅ Todo

A fast, keyboard-driven TUI todo list application built with Rust.

## Features

- Interactive terminal UI
- Add, remove, and list todos
- Mark tasks as complete
- Keyboard navigation

## Installation

```bash
git clone https://github.com/yourusername/todo
cd todo
cargo build --release
./target/release/todo
```

## Keybindings

| Key | Action |
|-----|--------|
| `a` | Add todo |
| `d` | Delete todo |
| `Space` | Toggle complete |
| `q` | Quit |
| `↑ / ↓` | Navigate |

## Requirements

- Rust 1.70+
